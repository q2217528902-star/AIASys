# AIASys 基础使用教程

> 当前版本: v0.4.23

这份教程覆盖当前主线最短上手路径。AIASys 当前按本地单用户产品设计，认证层使用 `local` / `none` 模式固定返回本地默认用户，不需要独立的登录流程，也不包含团队成员、邀请、在线协同和多人权限配置。

如果你还没有启动系统，请先完成 [部署说明](deployment.md) 或 [快速启动指南](guides/getting-started/QUICKSTART.md)。

## 1. 打开分析工作区

启动前后端后，打开：

```text
http://localhost:13000/workspace
```

`/workspace` 是当前主界面。页面不会强制你进入某个固定任务，你可以先恢复已有工作区，也可以新建一个长期任务工作区。

## 2. 理解界面布局

进入某个工作区后，主界面按「左侧 Activity Bar + 中间主画布 + 右侧聊天侧栏」组织：

```text
左侧 Activity Bar | 中间主画布 | 右侧聊天侧栏
```

**左侧 Activity Bar** 用于切换工作区视图和功能面板，支持拖拽调整顺序。默认从上到下依次是：当前工作区（文件管理）、全局工作区（跨工作区共享资源）、数据查询、文件搜索、专家协作节点、文件变更。点击图标展开对应面板，可以在里面直接操作。

其他功能入口（监控任务、终端、自动化任务、环境变量、能力管理、工作区设置）分布在工作区画布、顶部工具栏或右侧面板中。

**中间主画布** 未打开工作区时显示工作区首页（选择或创建工作区）；打开工作区后默认进入「当前工作区」面板，展示文件资产目录。点击资源节点（知识库、知识图谱、数据库等）会在画布区域或侧栏中打开对应的预览面板。

**右侧聊天侧栏** 是当前会话的协作上下文入口，包含对话区、当前工作区文件预览、资源预览、上下文与预算信息。对话只是推进任务的一个入口，不是唯一的交互方式。

## 3. 左侧 Activity Bar 各面板

### 当前工作区

中间主画布默认进入「当前工作区」面板。这里显示当前会话的文件资产目录，支持拖拽上传、新建文件/文件夹、搜索和右键操作。文件直接在侧栏中编辑或预览，支持 Markdown、JSON、CSV、Python、Jupyter Notebook 等多种格式。

### 全局工作区

`全局工作区`（左侧 Activity Bar 第二个图标）用来管理跨工作区共享的资源，包括全局知识库、数据库连接、图谱等。放在全局工作区里的文件在所有工作区里都能访问到，不需要复制。

### 数据查询

左侧 Activity Bar 的 `数据查询` 面板用于连接数据库并执行 SQL。支持选择内置 DuckDB 或外部数据库连接，查看表和字段，执行查询并把结果作为当前会话上下文使用。

右侧聊天侧栏会继续保留，方便你边看数据边和 Agent 对话。

### 文件搜索

左侧 Activity Bar 的 `文件搜索` 面板用于在工作区内搜索文件内容。支持全文检索和过滤。

### 监控任务与自动化任务

`监控任务` 和 `自动化任务` 不在默认 Activity Bar 中，而是通过工作区工具菜单、顶部工具栏或设置面板进入：

- **监控任务**：查看异步执行的长时间运行任务状态和输出
- **自动化任务**：配置单次、周期、Cron 和连续推进任务，支持绑定当前会话继续推进或每次触发新建会话

### 专家协作节点与协作专家

左侧 Activity Bar 的 `专家协作节点` 面板用于查看多 Agent 并行执行流程，主控 Agent 在最上层，每个子 Agent 是下面一个节点，颜色表示状态（运行中、完成、失败），点开节点可以看到完整的对话历史、工具调用记录和产出文件。

`专家协作节点` 面板用于查看多 Agent 并行执行流程；`能力管理`（管理 MCP/Skill/协作专家）不在默认 Activity Bar 中，需通过设置菜单或工作区工具菜单进入。系统提供代码专家（coder）、研究员（researcher）、审查员（reviewer）和通用执行者（worker）等协作专家候选，启用后主控 Agent 才能在运行时派发。

### 知识图谱可视化

在工作区的资源管理面板中打开图谱文件（如 `.graph.db`），即可进入知识图谱可视化界面。支持力导向布局、节点搜索、关系查询和原始数据查看。图谱工作台由 Pixi.js 渲染，d3-force 计算布局。

## 4. 在当前会话继续任务

右侧聊天侧栏代表当前会话。你可以在这里继续提问、上传当前轮附件、切换会话或新建会话。侧栏顶部显示当前会话的上下文占用和 token 预算。

## 5. 其他常用功能入口

还有一些实用功能通过工作区画布、顶部工具栏或设置菜单进入：

- **环境变量**：管理全局或当前工作区环境变量，Agent 执行代码时自动读取注入
- **能力管理**：查看和管理当前工作区可用的 MCP Server 和 Skill，按需启用或禁用
- **终端**：WebSocket 终端，直接连到工作区文件系统，Agent 可以在终端里执行命令、运行脚本
- **监控任务**：查看异步执行的长时间运行任务状态和输出
- **自动化任务**：创建单次、周期、Cron 和连续推进任务

## 6. 推荐第一条体验路径

第一次体验可以按这个顺序走：

1. 打开 `/workspace`。
2. 新建或选择一个工作区。
3. 点击左侧 `当前工作区` 图标查看文件列表，拖拽上传几个文件试试。
4. 点击左侧 `全局工作区` 图标查看跨工作区共享的资源。
5. 在右侧聊天侧栏输入第一个问题，开始和 Agent 对话。
6. 尝试点击左侧 `数据查询` 连接数据库，或通过工作区工具菜单/设置创建一个自动化任务。
