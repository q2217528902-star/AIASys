# 工作区模板

工作区模板是预设的工作区结构，包含文件布局、AGENTS.md 协作指南、示例代码、环境变量和推荐能力。创建新工作区时选择模板，一键完成初始化。

## 内置模板

系统预置 7 种模板：

| 模板 | 说明 |
|------|------|
| 官方默认 | 通用工作模板，预装平台、Canvas、UV 运行环境等基础 Skill |
| 空白工作区 | 从零开始，只预置通用 AGENTS.md 协作指南和基础目录结构 |
| PDF 翻译 | 一键翻译外文 PDF，预装 pdf2zh 翻译引擎，支持 Google 免费翻译和 OpenAI/Gemini 高质量翻译，输出单语版和双语对照版 |
| 数据分析 | 预置 data/、notebooks/、figures/、reports/ 目录，附带示例分析脚本和 pandas/matplotlib 依赖声明 |
| 论文精读 | 预置 papers/、notes/、synthesis/ 目录，附带结构化笔记模板和系统阅读方法论 |
| 知识管理 | 预置 sources/、wiki/、docs/ 目录，基于 LLM Wiki 模式的知识管理工作流 |
| 竞赛攻关 | 预置实验跟踪、baseline 管理、文献索引结构，附带示例竞赛数据 |

## 创建工作区时选择模板

创建新工作区时，模板选择器展示所有可用模板。选择模板后：

1. 系统展示模板的文件树预览，可勾选要导入的文件子集
2. 系统展示模板推荐的能力（Skill、MCP、协作专家），可勾选要安装的能力（required 项不可取消）
3. 创建工作区后，后端自动写入勾选的文件、复制 workspace_memory.md、应用环境变量、安装推荐能力

## 保存当前工作区为模板

将当前工作区的内容保存为新模板，供后续复用。保存时可以选择：

- 包含哪些文件和目录（支持类 gitignore 的排除规则，默认排除 `.aiasys/`、隐藏文件和缓存）
- 填写模板名称、描述、分类和图标
- 是否包含环境变量（默认关闭，防止敏感信息泄露）

保存后的模板写入用户全局工作区的 `.aiasys/templates/{template_id}/`，出现在模板列表中，创建新工作区时即可选择。

## 模板市场

模板市场提供系统内置模板的浏览和安装：

- 当前市场源为 `aiasys-builtin`（AIASys 内置）
- 支持按关键词搜索和按分类过滤
- 支持将内置模板安装到用户模板目录（安装后可像自定义模板一样管理）
- 已安装的模板会显示"已安装"，支持覆盖重装

市场架构预留了多源扩展能力，后续可接入第三方模板市场。

## 模板管理

模板管理界面支持：

- 查看所有已安装模板（系统内置 + 用户自定义）的列表和详情
- 删除用户自定义模板
- 内置模板不可删除

## 模板作用域与覆盖规则

- **系统内置模板**：存放于后端 `apps/backend/templates/`，只读，随系统更新
- **用户自定义模板**：存放于 `workspaces/{user_id}/global_workspace/.aiasys/templates/`
- **覆盖规则**：用户目录中的同名模板优先于系统内置模板
