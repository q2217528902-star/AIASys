# 文件管理

工作区内的所有文件和资源通过左侧 Activity Bar 的文件树进行管理。

![当前工作区文件面板](../../../images/readme/panel-current-workspace.png)

## 文件树

点击左侧 Activity Bar 的"当前工作区"图标打开文件树面板。

### 基本操作

- 点击文件夹左侧箭头展开/折叠子目录
- 点击文件名在主画布中打开文件
- 文件类型通过图标区分，不同类型使用不同图标：

| 文件类型 | 图标 | 说明 |
|----------|------|------|
| 图片（PNG/JPG/GIF/SVG/WebP） | 图片图标 + 缩略图 | 文件树中直接显示缩略图，点击可打开灯箱预览 |
| 表格（CSV/XLSX/XLS） | 表格图标 | |
| 演示文稿（PPT/PPTX） | 演示文稿图标 | |
| Notebook（.ipynb） | Notebook 图标 | 右键菜单有"在 Notebook 中打开"和"打开预览"两个入口 |
| 代码（PY/JS/TS/HTML/CSS/JSON） | 代码文件图标 | 右键菜单有"编辑文件"入口 |
| Canvas（.canvas） | 画布图标 | |
| 数据库（.db/.sqlite/.duckdb） | 数据库图标 | 按存放路径决定预览方式 |
| Markdown（.md） | 文本文件图标 | 右键菜单有导出子菜单（MD/DOCX/PDF） |
| 知识库资源 | 书籍图标 | |
| 知识图谱资源 | 网络图标 | |
| Memory 文件 | 文本文件图标 | `.aiasys/memory/` 下的 Markdown 文件 |
| 新文件 | 蓝色脉冲圆点 | 标记最近创建的文件 |
| 大型目录 | 警告色标签 | 如"Python 环境"、"Node 依赖"、"大型目录"，默认分页加载 |

### 右键菜单

在文件或文件夹上右键弹出上下文菜单，可用操作取决于文件类型：

| 操作 | 适用对象 | 说明 |
|------|----------|------|
| 打开 | 文件 | 在默认编辑器中打开 |
| 编辑 | 文本文件 | 在 CodeMirror 编辑器中打开 |
| 在画布中打开 | 图片/PDF | 在 Canvas 中打开预览 |
| 在画布中编辑 | 图片 | 在 Canvas 中编辑 |
| 文件历史 | 所有文件 | 查看版本历史和差异对比 |
| 导出 | 文本文件 | 导出为 Markdown/DOCX/PDF |
| 复制路径 | 所有 | 复制文件在工作区中的相对路径 |
| 下载 | 所有 | 下载文件到本地 |
| 移动 | 所有 | 移动到工作区内其他目录 |
| 删除 | 所有 | 删除文件或文件夹 |

### 多选模式

点击文件树顶部的"多选"按钮进入多选模式：

- 勾选需要操作的文件或文件夹
- 底部出现批量操作工具栏，支持：批量复制、批量移动、批量删除
- 再次点击"多选"或按 Esc 退出多选模式

### 目录操作

在文件夹上右键或在文件树空白处右键，可以：

- 新建文件：创建空白文件，需输入文件名
- 新建文件夹：创建子目录
- 粘贴：将剪贴板中的文件粘贴到当前位置
- 删除：删除文件夹及其全部内容

### 重目录折叠

以下目录会被自动识别为重目录，默认折叠且显示为简化视图：

- Python 虚拟环境目录（`venv`、`.venv`、`env`）
- Node.js 依赖目录（`node_modules`）
- Python 缓存目录（`__pycache__`、`.pytest_cache`）
- 系统配置目录（`.git`、`.aiasys`）

重目录不参与文件搜索和批量操作，减少误操作风险。

![文件搜索面板](../../../images/readme/panel-file-search.png)

## 文件上传

支持以下方式上传文件到当前工作区：

- 从本地文件管理器拖拽文件到文件树或主画布区域
- 在文件树空白处右键选择"上传文件"

上传进度在页面顶部横幅显示，支持同时上传多个文件。

## 文件编辑

AIASys 使用 CodeMirror 作为内置代码编辑器。

### 支持的语言

编辑器根据文件扩展名自动选择语言模式，支持以下语言的语法高亮：

CSS、HTML、JavaScript、JSON、Markdown、Python、SQL、XML、YAML

### 编辑状态

- 编辑器标签页标题旁显示圆点指示器：绿色表示已保存，橙色表示有未保存的修改
- 切换到其他文件或关闭标签页时，如有未保存修改会弹出确认提示
- 只读文件在编辑器中显示锁图标，不可编辑。只读来源包括：文件历史查看、挂载的只读目录

### 自动保存

编辑器默认开启自动保存，停止输入 2 秒后自动写入。可以在设置中关闭自动保存，改为手动 Ctrl+S。

## 文件预览

不同文件类型在主画布中有对应的预览方式：

| 文件类型 | 预览方式 |
|----------|----------|
| 图片（PNG/JPG/GIF/SVG/WebP） | 灯箱预览，支持缩放和旋转 |
| PDF | 内嵌 PDF 查看器，支持翻页和缩放 |
| Office 文档（DOCX/PPTX/XLSX） | 转换为 HTML 后内嵌预览 |
| CSV | 表格视图，支持排序和筛选 |
| SQLite 数据库（.db/.sqlite） | 数据库预览面板，显示表列表和行数 |
| DuckDB 数据库（.duckdb） | 数据库预览面板，显示表列表和行数 |
| Jupyter Notebook（.ipynb） | Notebook 执行界面 |
| Memory 条目 | 只读 Markdown 预览 |

## 文件历史

右键文件选择"文件历史"打开版本历史面板。

![文件变更历史面板](../../../images/readme/panel-file-changes.png)

### 版本列表

左侧面板按时间倒序列出所有历史版本，每条记录包含：

- 版本时间戳
- 编辑者标识（用户直接编辑或 Agent 编辑）
- 文件大小

### 版本操作

- 点击版本记录，右侧面板显示该版本的完整内容
- 点击"对比"按钮，右侧面板切换为差异视图，与当前版本进行逐行对比
- 点击"恢复到此版本"，将文件内容回退到选中版本。恢复操作本身也会生成一个新的历史版本

### 差异视图

差异视图使用统一差异（unified diff）格式：

- 绿色背景 + `+` 前缀：新增行
- 红色背景 + `-` 前缀：删除行
- 两侧均显示行号
- 未变化的行默认折叠，可点击展开

每个文件默认保留最近 50 个历史版本，单文件上限 2MB。超出限制时最旧的版本自动清理。

## 文件导出

支持将文本文件导出为以下格式：

- Markdown 文档：保留原始 Markdown 格式，导出为 `.md` 文件
- DOCX 文档：Markdown 转 Word 文档
- PDF 文档：Markdown 转 PDF

导出操作在右键菜单中触发，导出文件下载到本地。

## 资源节点

以下资源在左侧边栏以资源节点的形式出现，点击即可在主画布中预览：

- 知识库：显示文档列表，点击文档查看内容
- 知识图谱：显示图谱可视化
- 数据库连接：显示表列表，点击表名查看数据
- Memory 条目：显示记忆内容

资源节点与文件树共享同一侧栏区域，通过图标区分类型。
