# 全局控制面板

全局控制面板用于管理跨工作区的系统级配置，包括模型连接、环境变量、存储位置、能力市场、自动化任务和模板中心等。

## 打开方式

可以通过以下入口打开全局控制面板：

- **快捷键**：`Ctrl + ,`（Windows/Linux）或 `Cmd + ,`（macOS）。
- **系统托盘菜单**：桌面版点击托盘“设置”项可直接进入对应分类。
- **工作区上下文**：
  - 工作区首页的“打开设置”按钮。
  - 右侧会话侧栏顶部设置下拉菜单（工具配置、模型配置、执行环境）。
- **引导检查**：
  - 首次使用快速设置清单（SetupChecklist）。
  - 创建工作区前的环境前置检查（PreflightCheck）。
  - DockHeader 状态徽标（Shell 环境、被禁用的 AutoTask 等）。

从具体场景进入时，面板会自动定位到对应的设置分类（例如从“未配置模型”提示进入时直接打开“模型配置”）。

## 面板结构

面板采用左侧导航、右侧内容的布局：

- **左侧导航**：按功能分组，支持搜索、键盘方向键导航、分组展开/折叠。
- **右侧内容**：显示当前选中分类的表单、列表或管理界面。
- **关闭**：点击右上角关闭按钮、点击背景遮罩或按 `Esc` 即可关闭。

## 设置分类

| 分类 | 说明 |
|---|---|
| **能力管理** | 统一管理 Skill、MCP 连接器和协作专家的安装、启用与禁用。 |
| **我的默认配置** | 跨工作区共享的个人基线：工作说明、工具选择、加载策略、运行时参数和任务模型路由。 |
| **模型配置** | 配置 LLM 服务商连接信息和模型参数。 |
| **全局环境变量** | 对所有工作区生效的环境变量；工作区级别的同名变量会覆盖此处。 |
| **uv 包管理器镜像** | 配置 PyPI、Python 二进制和 uv 安装器的镜像源。 |
| **环境增强** | 查看 shell 环境状态并安装/下载可选的跨平台组件。 |
| **存储位置** | 配置新建工作区、全局资源和日志的默认落盘位置。 |
| **执行资源** | 管理 Python、Node.js、Docker 容器和注入变量。 |
| **自动化任务** | 查看和管理跨工作区的 AutoTask。 |
| **监控任务** | 查看和管理后台监控任务执行状态。 |
| **模板市场** | 浏览和安装系统内置工作区模板。 |
| **模板管理** | 浏览和管理已安装的工作区模板。 |
| **Token 消耗** | 查看跨会话的 LLM Token 消耗趋势与热力图。 |

## 保存方式

不同分类的保存节奏由对应面板自身控制：

- **即时保存**：例如全局环境变量在增删改后会立即调用后端接口保存。
- **手动保存**：例如模型配置、执行资源等需要点击面板内的“保存”或“应用”按钮。

关闭全局控制面板不会触发统一的全局保存；如果当前分类有未保存的修改，建议先完成保存再关闭，以免丢失。

## 导航与搜索

- 点击左侧导航切换分类；当前选中的分类会高亮显示。
- 使用搜索框可快速过滤分类名称和描述，匹配的关键词会在导航标签中高亮。
- 面板会记住最近一次活跃的分类；下次从通用入口（如快捷键、托盘菜单）打开且未指定具体分类时，会自动恢复到上次停留的分类。
- 支持键盘操作：
  - `↑` / `↓`：在可聚焦的导航项之间移动。
  - `←` / `→`：折叠/展开当前分组。
  - `Home` / `End`：跳到第一个/最后一个导航项。
  - `Esc`：关闭面板。
