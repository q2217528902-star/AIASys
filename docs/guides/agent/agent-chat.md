# Agent 对话与会话

> 当前版本: v0.3.9

本文档说明如何与 Agent 对话，以及会话管理的各项操作。

## 发起对话

进入工作区后，在右侧聊天侧栏底部的输入区输入内容，按 Enter 发送。Agent 收到消息后会先判断当前请求类型，再决定直接执行、只读核验或请求确认。

输入区支持多行文本，Shift+Enter 换行。输入区上方显示当前使用的模型名称，点击可切换模型。

### 请求类型

- 明确要求修改、创建、运行、测试、调用工具或处理文件时，Agent 会直接进入执行流程。
- 询问"你觉得怎么样"、"帮我评估一下"、"看看设计是否合理"这类审查、咨询和复盘请求时，Agent 优先做只读核验并给出判断，不会默认写文件、安装 Skill、启用运行环境或启动长任务。
- 涉及安装或启用 Skill、写入全局工作区、切换运行环境、更新系统基线、调整工具配置、创建 AutoTask、删除或覆盖文件、启动长时间后台任务时，按当前授权模式决策：manual 模式需要逐条确认；smart 模式按风险等级自动决策（低风险放行、高风险询问、极高风险拦截）；auto 和 full_auto 模式在硬安全边界内自动执行。

## 会话管理

每个工作区可以有多个会话，会话之间上下文隔离。会话列表显示在右侧侧栏顶部，按时间倒序排列。

### 创建会话

点击会话列表右侧的"+"按钮新建会话。新会话从空白上下文开始，不继承其他会话的历史。

### 切换会话

点击会话列表中的会话名称即可切换到该会话。切换后输入区和对话历史都会更新为对应会话的内容。

### 自动命名

新建会话时默认名称为"新会话"。当对话进行到一定轮次后，Agent 会根据对话内容自动生成一个概括性的会话名称，方便后续查找。

手动修改名称：点击会话名称进入编辑状态，修改后按 Enter 确认。

### 会话删除

右键点击会话名称，选择"删除会话"。删除后会话及其全部历史记录不可恢复。

## 模型选择

模型配置采用三层覆盖机制，优先级从低到高：

1. **全局默认**：在系统设置中配置，作为所有工作区的兜底模型
2. **工作区设置**：覆盖当前工作区的默认模型
3. **会话覆盖**：在当前会话中临时切换模型，不影响其他会话

切换方式：点击输入区上方的模型选择器，从已配置的模型列表中选择。当前会话生效的模型由优先级最高的层决定。

## 附件上传

在输入区左侧点击附件按钮，或直接拖拽文件到输入区，可以将文件作为对话上下文上传。

支持的附件类型：

- 文档：PDF、Word、Markdown、TXT
- 数据：CSV、JSON、Excel
- 代码：Python、JavaScript、TypeScript 等
- 图片：PNG、JPG、GIF、SVG

上传后文件会显示在输入区上方，Agent 会在执行时读取这些文件的内容。已上传的附件可以手动移除。

## 执行流

Agent 执行任务时，右侧侧栏会实时展示执行流。执行流以树状结构呈现：

- **根节点**：主控 Agent 的任务描述和状态
- **子节点**：Sub Agent 被派发执行子任务时的状态
- **工具调用**：每个节点下展开显示 Agent 调用了哪些工具（文件读写、Shell 执行、API 调用等）

节点颜色表示状态：

- 蓝色：运行中
- 绿色：已完成
- 红色：执行失败

点击节点可展开查看详细信息，包括工具调用的输入参数和返回结果。

## 富文本聊天

Agent 的回复支持多种富文本格式：

- **Markdown**：标题、列表、表格、链接、引用等标准 Markdown 语法
- **代码高亮**：代码块按语言自动语法高亮，支持复制按钮
- **数学公式**：使用 KaTeX 渲染行内公式和块级公式，支持 LaTeX 语法
- **ECharts 图表**：Agent 生成的数据可视化图表直接内嵌在回复中
- **Mermaid 图表**：流程图、时序图、甘特图等通过 Mermaid 渲染
- **思考过程**：Agent 的推理过程以可折叠区块显示，标题显示"思考中"，完成后可展开查看

## 图片阅读

Agent 可以通过 ReadMedia 工具读取和分析图片内容。当用户在对话中上传图片，或在工作区中有图片文件时，Agent 可以：

- 识别图片中的文字（OCR）
- 描述图片内容和场景
- 分析图表和数据可视化
- 提取图片中的结构化信息

Agent 会在需要时自动调用 ReadMedia，不需要用户显式指定。

## PDF 翻译

Agent 支持将外文 PDF 文档翻译为中文。翻译保持原文档的版式结构，包括段落、表格、标题层级。

使用方式：在对话中告诉 Agent "把这份 PDF 翻译成中文"，Agent 会自动调用 PDF 翻译能力完成翻译。

翻译结果以 Markdown 文件形式保存到当前工作区。

## AskUser 确认

当 Agent 在执行过程中遇到需要用户决策的场景时，会暂停执行并弹出确认对话框。典型场景包括：

- 需要用户确认某个操作方向
- 多个可选方案需要用户选择
- 执行结果不明确需要用户判断

对话框中显示 Agent 的问题和可选操作。用户可以：

- 选择"确认"让 Agent 继续
- 选择"拒绝"让 Agent 换方向
- 输入补充信息后确认

确认对话框在右侧侧栏中弹出，不影响其他界面操作。

## 图片灯箱

对话中的图片支持点击放大查看。点击任意图片后弹出灯箱，显示原图尺寸。灯箱支持：

- 鼠标滚轮缩放
- 拖拽平移
- 点击背景或关闭按钮退出

多张图片时支持左右切换浏览。
