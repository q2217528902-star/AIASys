# Canvas 画布

Canvas 是 AIASys 内置的可视化画布编辑器，基于 JSON Canvas 标准格式，支持无限画布上的节点与连线编辑。

![Canvas 工作流画布演示](../../../images/readme/demo-canvas-workflow.png)

## JSON Canvas 格式

画布文件使用 `.canvas` 扩展名，内部遵循 [JSON Canvas](https://jsoncanvas.org/) 开放标准。每个 `.canvas` 文件是一个 JSON 文档，包含 `nodes` 数组和 `edges` 数组。这个标准格式让画布文件可以被其他兼容工具打开，不锁定在 AIASys 内。

## 创建画布

在文件树工具栏点击"新建"按钮，选择"Canvas 画布"，输入文件名（自动添加 `.canvas` 后缀）。新画布会在当前工作区目录下创建，并在主画布区域自动打开。

也可以在 Agent 对话中直接让 Agent 创建画布，Agent 会生成符合 JSON Canvas 规范的 `.canvas` 文件。

## 打开画布

在文件树中点击任意 `.canvas` 文件，画布编辑器会在主画布区域打开。支持同时打开多个画布，每个画布显示为独立标签页。

## 节点操作

### 创建节点

三种方式创建节点：

- 双击画布空白区域，在点击位置创建一个默认文本节点
- 从工具栏拖拽节点类型到画布上
- 右键画布空白区域，从上下文菜单中选择"添加节点"

支持的节点类型：文本（text）、文件（file）、链接（link）、图片（image）。

### 编辑节点

- **移动**：拖拽节点到任意位置
- **文本编辑**：双击节点进入编辑模式，直接输入内容。按 `Esc` 或点击画布其他区域退出编辑
- **调整大小**：拖拽节点右下角的缩放手柄
- **删除**：选中节点后按 `Delete` 或 `Backspace` 键，或右键选择"删除节点"
- **多选**：按住 `Shift` 点击多个节点，或在画布上拖拽框选。多选后可批量移动或删除

### 属性面板

选中节点后，右侧属性面板显示当前节点信息：

- **类型**：节点的类型（文本/文件/链接/图片）
- **颜色**：节点背景色，从预设色板中选择

修改后即时生效，自动保存。

## 连线操作

### 创建连线

从节点的输出端口（右侧圆点）拖拽到另一个节点的输入端口（左侧圆点），释放鼠标即创建一条有向连线。

### 编辑连线

- **添加标签**：双击连线，输入标签文字
- **编辑标签**：双击已有标签进行修改
- **删除连线**：选中连线后按 `Delete` 键，或右键选择"删除连线"

连线默认从源节点指向目标节点，标签显示在连线中间位置。

## 平移与缩放

| 操作 | 方式 |
|------|------|
| 放大/缩小 | 鼠标滚轮 |
| 平移画布 | 按住空格键 + 拖拽鼠标，或鼠标中键拖拽 |
| 适应窗口 | 点击工具栏"适应画布"按钮 |
| 缩放至 100% | 点击工具栏缩放百分比标签 |

工具栏右下角显示当前缩放比例，可点击快速重置。

## 工具栏

画布顶部工具栏提供以下快捷操作：

- 节点类型按钮：点击后在画布上放置对应类型节点
- 缩放控制：放大、缩小、适应画布
- 撤销/重做：支持画布操作的撤销与重做

## 上下文菜单

右键点击画布或节点弹出上下文菜单：

| 位置 | 菜单项 |
|------|--------|
| 画布空白 | 添加节点、粘贴 |
| 节点 | 编辑、复制、删除节点、添加连线 |
| 连线 | 编辑标签、删除连线 |

## 文件附件

可以将工作区中的文件附加到画布节点上。在属性面板中点击"附加文件"，从文件选择器中选择目标文件。附加后，节点上会显示文件名，点击可预览文件内容。

适用场景：将参考资料、数据文件、分析结果关联到画布上的对应节点。

## 自动保存

所有画布操作（创建节点、移动、编辑、连线、删除）均自动保存到 `.canvas` 文件，无需手动保存。撤销/重做栈也会在自动保存后保留。

## Agent 交互

Agent 可以通过系统内置的文件工具直接读写 `.canvas` 文件。具体能力：

- **读取**：Agent 可以读取画布的完整 JSON 结构，包括所有节点和连线
- **写入**：Agent 可以创建新画布或修改现有画布
- **批量操作**：Agent 可以一次性添加多个节点和连线，适合自动生成思维导图、大纲等结构化内容

此外，系统内置了 Canvas Skill（`aiasys-canvas-skill`），提供画布脚本编辑和 JSON Canvas 格式校验能力。Agent 在需要批量修改画布或校验画布格式时，可以加载此 Skill。

## 使用场景

- **头脑风暴**：自由放置想法节点，用连线组织逻辑关系
- **思路整理**：将碎片想法可视化，逐步归类分组
- **项目规划**：分解任务节点，用连线表示依赖关系
- **文章大纲**：节点作为章节标题，连线表示段落间的逻辑顺序
