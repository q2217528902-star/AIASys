# Changelog v0.3.2 - 2026-04-10

## li-xiu-qi

### 重构 MCP 市场与 Skill 市场主链

#### 问题描述
- 现象：外部 MCP、Skill 市场、工作区挂载和实际启用关系长期分散在多处 UI，入口跳转和数据回读都不统一。
- 原因：市场目录、工作区配置和会话级启用链路没有围绕工作区主入口收口。

#### 改动内容
- 后端新增外部 MCP / Skill 市场模型与服务，并扩展工作区 API 以支持市场预览、安装和回读。
- 前端重构 MCP 市场和 Skill 市场面板，新增外部目录预览、工作区对话框集成和生命周期回归覆盖。
- 工作区设置、Skill 市场、左侧导航和右侧工作区上下文现在围绕统一的工作区入口协作。

### 分离“我的默认”和“工作区设置”

#### 问题描述
- 现象：左下角齿轮和工作区入口打开的是同一套可见分组，用户很难分清默认层和工作区层。
- 原因：`WorkspaceSettingsDialog` 同时承接了两种入口语义，但没有在导航层明确分开。

#### 改动内容
- 侧栏底部齿轮改为只打开“我的默认”，工作区入口只打开“工作区设置”。
- 两个入口继续复用同一套对话框壳层，但左侧分组和默认焦点已经按作用域分离。
