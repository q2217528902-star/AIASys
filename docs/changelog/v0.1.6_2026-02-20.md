# 版本更新记录 v0.1.6

**更新日期**: 2026-02-20

## li-xiu-qi

### 优化与改进 (Improvements)

- **前端设计系统统一**:
  - 引入 Inter + Noto Sans SC + JetBrains Mono 字体栈，替换系统默认字体
  - 启用 font-smoothing 和 font-feature-settings 提升渲染质量
  - 全局 prose 组件样式覆盖（~130 行），统一标题/列表/代码/表格/blockquote 排版
  - 行高从默认提升到 1.75，适配中文阅读

- **色调统一（50+ 组件）**:
  - 修复 `index.css` 暗色模式变量（rgb → oklch）
  - 替换所有硬编码 hex 色值和 Tailwind 原始色为语义 token
  - 删除 `App.css` Vite 模板残留

- **右侧边栏重设计（MultiTaskSidebar）**:
  - 收起状态从 8px 窄条改为 36px 宽条 + 垂直文字"执行空间" + hover 高亮
  - 移除冗余 TaskStatusBar（信息已在 TaskTabs 图标中体现）
  - SidebarHeader 紧凑化（节省 ~12px 垂直空间）
  - WorkspaceArtifacts 预览面板新增关闭按钮
  - FooterStatus 增强为运行/完成/失败计数 + 事件数显示

- **聊天消息样式优化**:
  - 工具调用按钮重设计：工具 icon + 双行布局 + 语义 token
  - 思考过程块重设计：圆角卡片容器 + Brain icon 动画 + 旋转箭头
  - FinalAnswerBlock 移除 `w-[600px]` 硬编码宽度限制
  - ToolBlock 展开内容移除 `slate-*` 硬编码颜色

### 修复 (Bug Fixes)

- **路由 trailing slash 问题**: `parsePathParams` 未处理 `/analysis/abc123/` 格式
- **navigate 引用不稳定**: 未使用 `useCallback`，导致 useEffect 重复执行
- **useAgentStream stale closure**: `state.taskId` 闭包过期，改用 ref 追踪
- **loadHistorySessions 缺少 loading 状态**: 未设置 `isLoadingHistory` 标志

### 文档完善 (Documentation)

- **前端设计规范**: 新增 `topics/frontend-design-standards.md`
  - 设计哲学、字体系统、颜色 token 规范、prose 排版规范
  - 组件设计模式、交互规范、禁止事项、审查清单

- **删除无用兼容代码**: 移除前端 `subagent_event` 旧格式兼容代码

- **新增 Skill**: 创建 `code-cleanup-compat` skill
