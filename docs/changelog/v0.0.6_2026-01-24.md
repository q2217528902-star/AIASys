# 版本更新记录 v0.0.6

> 历史说明:
> 本文档记录早期工程阶段的前后端结构与开发方式；
> 当前项目目录、代理配置与运行方式已有多轮演进，请以现行 `README.md` 和产品文档入口为准。

更新日期: 2026-01-24

## 贡献者ID li-xiu-qi

### 新增功能
- 后端引入 Celery 并支持两种运行模式
  - Eager 同步模式 用于本地调试
  - Redis 异步模式 用于生产形态
- 增加示例任务与模式自检脚本

### 优化与改进
- 主对话支持分段渲染
  - 支持 thought 与 final_answer 混排
  - 将 Host 的工具调用与输出移动到主对话并支持折叠
- Worker 运行提示增强
  - 主对话显示 Worker 状态摘要并支持点击跳转到右侧任务 Tab
  - Worker 任务按 worker_name 拆分为多个 Tab
- 工程配置优化
  - Vite 代理目标支持通过 VITE_API_TARGET 配置

### 修复
- 修复流式输出中 think 分段重复或覆盖的问题
- 修复任务结束后加载态不停止的问题
- 右侧侧边栏任务列表过滤 Host 任务 仅展示 Worker
- UI 文案清理 去除 emoji 前缀标记
- Frontend Build: 移除了 frontend/Dockerfile 中的 npm mirror 配置 修复 lefthook 包 404
- Backend Import: 修复容器启动时 ModuleNotFoundError No module named Agentbackend
  - Agentbackend/Dockerfile WORKDIR 调整为 /Agentbackend
  - PYTHONPATH 设置为 /
  - docker-compose.yml 卷挂载路径对齐

### 文档完善
- 新增前端侧文档
  - 流式渲染实现说明
  - SSE 协议与类型对齐规范
  - MultiTaskSidebar 交互逻辑说明
  - 会话与上下文同步机制
- 新增后端侧文档
  - Celery 任务队列指南
  - 异步任务架构设计方案

### 其他
- 合并 dev 分支最新修改
- 更新 docker-compose.yml 确保前后端 Redis Redis Commander 协同工作
