# 版本更新记录 v0.1.4

**更新日期**: 2026-02-02

## li-xiu-qi

### 新增功能 (Features)

- **中文字体支持优化**:
  - 添加思源黑体 (Noto Sans CJK SC) 字体文件到本地目录
  - 自动检测并加载挂载的本地字体文件
  - Docker 镜像配置中文字体安装
  - matplotlib 自动配置中文字体，无需手动设置

- **版本号管理规范**:
  - 建立版本号管理规范文档
  - 定义正式版本和开发细分版本规则
  - 前后端版本统一更新流程

### 优化与改进 (Improvements)

- **Docker Sandbox 字体加载**:
  - gateway.py 支持从挂载目录加载本地字体
  - kernel_interpreter.py 自动检测 Noto 字体可用性
  - 改进字体配置提示信息

- **AI 提示词优化**:
  - 更新 data_analysis_team.py 提示词
  - 说明字体已自动配置，无需手动干预

### 文档完善 (Documentation)

- **新增文档**:
  - `docs/guides/version-management-guide.md`: 版本号管理规范
  - `apps/backend/docs/05_guides/中文字体配置指南.md`: 字体配置说明
  - `docs/guides/README.md`: Guides 目录索引

- **文档索引更新**:
  - 更新 `AGENTS.md` 添加版本管理和字体配置索引
  - 更新 `apps/backend/docs/05_guides/README.md` 添加字体指南索引
