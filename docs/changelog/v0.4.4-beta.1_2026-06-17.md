# Changelog v0.4.4-beta.1 - 2026-06-17

## li-xiu-qi

### 新增功能 (Features)

- **Windows Shell 环境增强面板**
  - 全局设置新增「环境增强」入口，自动检测 Git Bash / WSL / busybox-w32 / Git / fnm / uv 的可用性。
  - 未安装组件提供官方下载链接；busybox-w32 支持一键自动下载到用户数据目录。
  - 影响文件：
    - `apps/backend/app/services/shell_environment.py`
    - `apps/backend/app/api/routes/system.py`
    - `apps/web/src/components/settings/ShellEnvironmentPanel.tsx`
    - `apps/web/src/components/settings/global-settings/index.tsx`
    - `apps/web/src/config/api.ts`

- **busybox-w32 作为 Windows shell fallback**
  - Windows 解释器优先级扩展为：Git Bash → WSL → busybox-w32 → PowerShell → CMD。
  - busybox-w32 使用 ash 语义执行简单 POSIX 命令，路径保持 `C:/foo/bar` 风格。
  - 影响文件：
    - `apps/backend/app/services/shell_executor.py`

- **Agent system prompt 按 shell family 分段注入**
  - 根据当前实际解释器（posix / wsl / busybox / powershell / cmd）给 Agent 注入不同的命令与路径适配提示。
  - 影响文件：
    - `apps/backend/app/services/agent/runtime_backends/aiasys/session.py`

- **Shell 工具 interpreter 参数增强**
  - 支持显式指定 `bash`、`wsl`、`busybox`、`powershell`、`cmd`。
  - 支持常见别名（`pwsh`、`sh`、`ash`、`wsl2` 等）和大小写不敏感。
  - 支持传入可执行文件绝对/相对路径（如 `C:\Program Files\PowerShell\7\pwsh.exe`）。
  - 影响文件：
    - `apps/backend/app/services/shell_executor.py`
    - `apps/backend/app/agents/tools/shell_tool.py`

- **第三方许可证声明**
  - 新增 `LICENSES/` 目录，包含 fnm（GPL-3.0）、busybox-w32 / Git for Windows（GPL-2.0）、uv / sqlite-vec 等组件的许可证文本与来源说明。
  - 桌面安装包打包时自动附带 `LICENSE` 和 `LICENSES/`。
  - 影响文件：
    - `LICENSES/*`
    - `apps/desktop/build/license.txt`
    - `apps/desktop/package.json`

- **Windows 环境快速验证脚本**
  - 新增 `scripts/dev/verify-windows-shell-env.ps1`，可一键检测当前 Windows 上各 shell 组件可用性并测试 busybox-w32。

### 优化 (Improvements)

- **环境检测增强**
  - Git Bash 检测增加 Windows 注册表兜底，处理 git 不在 PATH 但已安装的场景。
  - fnm / uv 检测优先读取桌面端注入的环境变量，探测不到时扫描 `vendor/` 目录。
  - `/api/system/shell-environment` 增加 30 秒 TTL 缓存，安装 busybox-w32 后自动失效。

### 测试 (Tests)

- 新增 `tests/test_shell_environment.py`，覆盖环境检测报告、推荐 family、guidance 生成。
- 新增 `tests/test_shell_executor.py`，覆盖 interpreter 别名、路径解析、大小写不敏感、异常分支。
