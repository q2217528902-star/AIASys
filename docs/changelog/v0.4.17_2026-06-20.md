# Changelog v0.4.17 - 2026-06-20

## ke

### 新增功能 (Features)

- **会话对话导入导出闭环（调试场景）**
  - 导出会话对话时直接读取 `history.json` 原始消息，不再经过前端展示管道，保留完整 LLM 上下文
  - 从 execution journal 回填被压缩成占位符的 tool 结果，方便他人导入后查看完整调试信息
  - 导出 JSON 为单个文件，同时包含原始 `content` 与 `display_content`，兼顾恢复与展示
  - 新增 `POST /api/sessions/{user_id}/import` 端点，支持将导出的对话 JSON 导入到指定工作区
  - 导入时自动生成新的 session_id 与 conversation_id，写入 `history.json` 并重建 `display_history.jsonl`
  - 前端对话面板新增「导入对话」按钮，支持选择 JSON 文件上传，导入成功后刷新工作区列表
  - 影响文件：
    - 后端：`apps/backend/app/services/export/session_export_service.py`、`apps/backend/app/services/export/session_import_service.py`、`apps/backend/app/services/export/__init__.py`、`apps/backend/app/api/routes/sessions_exports.py`
    - 前端：`apps/web/src/config/api.ts`、`apps/web/src/lib/api/sessions.ts`、`apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/WorkspaceConversationPanel.tsx`、`apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/DockHeader.tsx`、`apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/ConversationDock.tsx`、`apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/MainContent.tsx`、`apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/types.ts`、`apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/index.tsx`
