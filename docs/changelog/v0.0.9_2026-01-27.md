# 版本更新记录 v0.0.9

**更新日期**: 2026-01-27

##  li-xiu-qi

###  新增功能 (Features)
- **生产环境自动化部署体系**:
  - 新增 `docker-compose.prod.yml`，集成 API、Worker、Redis、PostgreSQL、Nginx 全栈环境。
  - 新增 `deploy_image.py` 和 `start_prod.py`，支持一键构建镜像并启动生产容器。
  - 为前后端新增 `Dockerfile.prod` 生产级配置，前端采用 Nginx 进行静态资源托管与反向代理。
- **会话持久化与状态恢复**:
  - 后端实现 `history_router.py` 会话历史接口。
  - 前端引入 `sessionRestore.ts` 逻辑，支持刷新页面后自动恢复会话元数据、上下文及代码运行状态。
- **工作区侧边栏 UI 深度重构**:
  - 实现 `FileTreeView` 树形目录组件，支持多级文件夹展示与交互。
  - 实现 `WorkspaceArtifacts` 组件，优化了对生成的产物文件的管理。
- **Jupyter Notebook (.ipynb) 深度预览支持**:
  - 新增 `NotebookPreview` 组件，支持交互式渲染 Notebook 内容（包括代码块、Markdown 及图片/文本输出）。
  - 在文件树中为 `.ipynb` 文件集成专用图标，并支持在工作区内联预览。
- **全方位空间资产管理 (Workspace Modal)**:
  - 重构并统一了「工作空间」弹窗与侧边栏的展示逻辑，支持内联预览所有资产文件。
  - 新增「图片快速预览」条，支持在弹窗顶部预览当前会话的前 5 张产物图，并提供 "+N" 样式展示。
  - 优化误触防护逻辑，将文件下载/打开动作为 Hover 触发的主动操作。

###  优化与改进 (Improvements)
- **Agent 回复展示优化**:
  - 改进了 Thought Block（思考块）的 UI 设计，层级更清晰，交互更顺滑。
  - 强化了 Data Analysis Agent 的系统指令，提升了处理大规模数据集时的稳定性。
- **文件预览交互增强**:
  - 修复了 `DataAnalysisPage` 中 props 传递缺失导致的预览内容加载失败问题。
  - 优化了 `FilePreviewPanel` 的样式，统一了代码、文本及 Notebook 的深色主题外观。
- **系统清理**:
  - 删除了废弃的 `start_backends.bat` 和 `start_backends.sh` 脚本。
  - 更新了全局 `.gitignore`，规范化了生产部署环境的忽略规则。
- **UI 细节优化**:
  - 修复多任务侧边栏中 Worker 名称显示冗余的问题（去除重复的 `[Worker] worker` 标签）。
  - 优化 Agent 流式输出展示，自动过滤 `TASK:`、`WORKER:` 等内部提示词前缀，保持界面清爽。

###  文档完善 (Documentation)
- **自动化部署手册**: 新增 `docs/AUTOMATED_DEPLOYMENT.md`，指导用户如何使用脚本进行生产部署。
- **生产环境操作指南**: 新增 `docs/DEPLOYMENT_GUIDE.md`。
