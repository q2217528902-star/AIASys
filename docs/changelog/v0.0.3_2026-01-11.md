# 版本更新记录 v0.0.3

**更新日期**: 2026-01-11

##  li-xiu-qi

###  新增功能：工作空间 (Workspace)
- ** 会话文件与报告查看**：新增了工作空间功能，允许用户浏览历史会话生成的所有文件，包括图表、报告、数据表格和 Notebook。
  - **功能入口**：左侧侧边栏新增 "工作空间" 按钮。
  - **文件浏览**：支持按历史会话查看文件，支持递归展示子目录，支持多种文件预览与筛选。
- ** 后端接口升级**：`/api/files/list` 支持递归读取子目录，以支持复杂任务生成的嵌套文件结构。

###  前端 UI 优化 (Frontend UI Optimization)
- ** 布局调整**：
  - **模式选择器**：从左侧边栏移至主内容区域顶部导航栏，采用简约风格。
  - **右侧边栏**：从绝对定位改为固定三栏布局，不再覆盖输入区域，保留拖拽功能。
- ** 样式优化**：全局滚动条美化，采用浅灰色圆角设计，兼容 Firefox。

###  Bug 修复 (Bug Fixes)
- **React DOM 同步错误修复**：修复了流式输出过程中 `dangerouslySetInnerHTML` 导致的 DOM 同步冲突，涉及 `AiResponseCard.tsx` 等多个组件。
- **运行状态判断逻辑修复**：更新了 `isRunning` 计算逻辑，确保多任务（Workforce）执行期间模式选择器能正确禁用。

###  系统安全增强 (System Security Enhancement)
- **OWL Agent 提示词优化**：在指挥者和执行者的系统提示词中明确添加了禁止自动安装依赖（如 `pip install`）的指令，以防止 Agent 尝试修改运行环境。
