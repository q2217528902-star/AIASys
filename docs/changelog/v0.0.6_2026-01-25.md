# 版本更新记录 v0.0.6

更新日期: 2026-01-25

## 贡献者ID li-xiu-qi

### 新增功能
- 身份验证体系完全重构
  - 移除 Supabase 依赖，整体迁移至本地 JWT 验证
  - 实现 BackendAuthService 对接本地 `/api/auth/login` 与 `/api/auth/register`
- 路由保护与自动跳转
  - 实现受保护页面的路由守卫，未登录自动跳转 `/login`
  - 支持登录后自动重定向回原访问页面

### 优化与改进
- UI 基础信息展示优化
  - 侧边栏及布局 footer 支持显示当前登录用户邮箱
  - 增加用户头像默认图标
- 后端接口权限校验
  - 统一 API 请求 Header 注入，支持 `Authorization: Bearer <token>` 规范

### 修复
- 修复流式输出导致的 UI 崩溃问题 (NotFoundError: Failed to execute 'removeChild')
  - 实现流式内容缓冲机制 (Buffer Mechanism)：通过 50ms 防抖批量更新，避免高频 setState 导致的 DOM 冲突
  - 应用 `useDeferredValue`：对聊天段落 (Segments) 进行延迟渲染，减轻 React 渲染压力
  - 稳定 React Key：使用稳定的索引 key 代替动态内容 key，防止流式更新过程中的元素重复挂载
  - 优化 `PlanningPlaceholder`：移除 JS 状态驱动的动画，改为纯 CSS 动画，彻底解决组件卸载时的异步状态残留问题
- 修复 React 运行时 DOM 操作导致的 removeChild 崩溃问题
  - 修复 ToolCard 组件由 innerHTML 引起的渲染冲突，改为状态驱动界面
  - 修复文件导出逻辑中 link 移除时的 parentNode 判定，增强操作健壮性
- 全局错误隔离
  - 引入顶层 ErrorBoundary 错误边界，防止组件崩溃导致整个网页白屏

### 文档完善
- 新增后端接口文档
  - `Agentbackend/docs/API_Authentication.md`: 详细定义了本地验证接口、JWT 格式及权限声明规范
