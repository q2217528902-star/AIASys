# Changelog v0.1.2 - 2026-01-30

##  li-xiu-qi

###  新增功能

#### 上下文压缩 (Context Compaction) - 核心功能
- **三层上下文压缩机制**
  - 解决长对话场景下的 Token 限制问题
  - 智能压缩历史上下文，防止上下文爆炸
  - 新增 `CompactionManager` 管理器协调压缩流程

- **压缩配置模块**
  - `CompactionConfig` - 压缩策略配置
  - `CompactionEstimator` - Token 使用量估算
  - `PromptBuilder` - 压缩后提示词构建

- **压缩 Agent 实现**
  - `CompressedCodeAgent` - 支持压缩的代码执行 Agent
  - `CompressedToolCallingAgent` - 支持压缩的工具调用 Agent
  - 集成到 `DataAnalysisTeam` Host-Worker 架构

- **新增配置项**
  - `compaction_offload_dir`: 压缩数据卸载目录 (`storage/compaction`)

#### Agent 技能库扩展
- **vercel-composition-patterns** - React 组合模式
  - Compound Components 模式
  - Render Props 和 Context Provider 最佳实践

- **vercel-react-native-skills** - React Native 开发
  - Expo 和原生模块最佳实践
  - 移动端性能优化指南

- **web-design-guidelines** - 网页设计规范
  - 可访问性 (A11y) 检查清单
  - UI/UX 最佳实践

- **smolagents 相关技能**
  - `smolagents_instructions` - smolagents 使用指南
  - `smolagents-tutorial-builder` - 教程构建工具

###  优化与改进

#### 目录结构重构
- **Agent 模块重组**
  - `app/smol_agents/` → `app/agents/`

  - 保持向后兼容的导入路径

- **模块划分优化**
  - `agents/core/` - 核心 Agent 实现
  - `agents/compaction/` - 上下文压缩模块
  - `agents/skills/` - Agent 技能库
