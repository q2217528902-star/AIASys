# Changelog v0.4.8 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **对话历史丢失（history.json 路径不一致）**
  - 现象：用户刷新页面后对话内容消失，API 返回 0 条消息
  - 根因：`sync_messages_to_history` 写入到 session 目录，但 `get_session_history` 从 workspace 目录读取
  - 修复：`get_session_history` 通过 `workspace_registry.get_session_dir()` 获取正确的 session 目录路径
  - 影响文件：`app/services/agent/mixins/history.py`

- **StepFun API 400 input_invalid（上下文压缩后）**
  - 现象：多轮对话后 StepFun API 间歇性返回 400 `input_invalid`
  - 根因：`clear_old_tool_results` 将 `tool_calls.function.arguments` 替换为非 JSON 文本，违反 OpenAI API 规范
  - 修复：替换为 `json.dumps({"_cleared": true, "_original_len": N})` 保持 JSON 有效
  - 影响文件：`app/services/agent/compaction.py`
