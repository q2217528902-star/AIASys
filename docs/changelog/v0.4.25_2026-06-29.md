# Changelog v0.4.25 - 2026-06-29

## Clannad47

### 修复暗黑模式默认模型下拉菜单显示异常

#### 问题描述
- 现象：全局控制面板的模型配置页中，暗黑模式下打开默认 Chat 模型或默认 Embedding 模型下拉框时，下拉菜单显示为白色背景，文字对比异常。
- 原因：默认模型区域使用原生 `select`，浏览器原生 `option` 弹层未跟随项目暗色主题变量。

#### 改动内容
- 将默认 Chat / Embedding 模型选择器替换为项目统一的 Select 组件。
- 使用内部哨兵值表示“未设置”，保存时继续映射为 `null`，保持原有默认模型配置语义不变。

#### 文件变更
- `apps/web/src/components/settings/llm-config/index.tsx`: 默认模型选择器改用统一 Select 组件。
