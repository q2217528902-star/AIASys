# Changelog v0.1.1 - 2026-01-30

##  li-xiu-qi

###  新增功能

#### 前端交互体验优化
- **全局快捷键支持**
  - `Esc` - 关闭右侧执行空间边栏
  - `Ctrl/Cmd + B` - 切换左侧历史边栏
  - `Enter` - 发送消息，`Shift + Enter` - 换行

- **删除会话确认对话框**
  - 在左侧历史边栏删除会话时增加二次确认
  - 防止误触删除重要会话

- **输入框自动调整高度**
  - 根据内容自动调整高度（60px-240px）
  - 支持多行输入，提升长文本输入体验

###  优化与改进

#### 前端代码质量
- **统一键盘事件处理**
  - 修复 `ChatInput.tsx` 使用 `onKeyUp` 与其他组件不一致的问题
  - 统一使用 `onKeyDown` 处理键盘事件

- **统一导航方式**
  - 修复 `DesignSidebar.tsx` 直接使用 `window.location.href`
  - 统一使用全局 `appNavigate` 函数进行导航

- **性能优化**
  - 缓存 `MarkdownIt` 实例，避免每次渲染重复创建

- **用户反馈优化**
  - 停止任务后显示 "任务已停止" toast 提示
  - 导出 Word 时使用 toast 替代 `alert` 提示成功/失败
