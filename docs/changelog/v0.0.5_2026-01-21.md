# 版本更新记录 v0.0.5

**更新日期**: 2026-01-21

##  贡献者ID
- li-xiu-qi

###  新增功能 (Features)
- **smolagents 深度集成**:
  - 全面引入 HuggingFace `smolagents` 库作为底层执行引擎，实现 `CodeAgent` 代码编写与执行能力。
  - 重构 `jupyter_executor.py` 以适配 smolagents 的代码执行接口，支持更强大的数据分析能力。
- **Host-Worker 协作架构**:
  - 实现基于多线程与共享队列的流式协作机制，确保 Host 能即时透传 Worker 的执行细节。
- **动态任务 Tab 系统**:
  - 前端侧边栏引入动态路由，自动为每个 Worker 创建独立的执行视图 Tab，保持主对话流整洁。

###  优化与改进 (Improvements)
- **单模式架构演进**:
  - 简化交互模型，锁定为单一的 "Host 智能中控" 模式，将原 `CodeExecutorTestPage` 升级为 `DataAnalysisPage`。
- **流式通信协议对齐**:
  - 统一适配 `/api/smol` 接口与标准的 `ProtocolEvent` 事件流，实现打字机式实时渲染。
- **代码库规范化**:
  - 将旧版 Owl/Workforce 相关代码迁移至 `archive/` 目录。
  - 为所有核心目录补充中文 README，明确文件组织规范。

###  修复 (Bug Fixes)
- **视觉捕获修复**: 修复了 Jupyter 执行器图片渲染失效的问题，实现对 `display_data` 消息的自动监听。
- **Tab 排序逻辑**: 优化了侧边栏 Tab 算法，确保 " Host" 始终置顶。
