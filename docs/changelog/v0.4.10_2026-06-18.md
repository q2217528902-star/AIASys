# Changelog v0.4.10 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **桌面构建版本号不与 tag 同步**
  - 现象：CI 构建产物文件名始终显示 `0.4.4-beta.1`，而非实际 tag 版本号（如 v0.4.9）
  - 根因：CI workflow 没有在构建前从 tag 同步版本号到 `package.json`/`pyproject.toml`
  - 修复：CI 中添加 `Sync version from tag` 步骤，从 tag 提取版本号更新三个包
  - 本地版本号同步更新到 `0.4.9`
  - 影响文件：`.github/workflows/ci-desktop.yml`、`apps/desktop/package.json`、`apps/web/package.json`、`apps/backend/pyproject.toml`
