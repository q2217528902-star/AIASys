# Changelog v0.4.6 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **SOCKS 代理环境崩溃**
  - 现象：环境中有 `ALL_PROXY=socks5://...` 时，httpx 尝试使用 SOCKS 代理但缺少 socksio 包导致 `ModuleNotFoundError`
  - 修复：`pyproject.toml` 中 `httpx>=0.28.1` 改为 `httpx[socks]>=0.28.1`，自动引入 socksio 依赖
  - 影响文件：`apps/backend/pyproject.toml`、`apps/backend/uv.lock`

- **CI ruff format/lint 失败**
  - 现象：v0.4.5 合并后 CI 的 `ruff format --check app/` 失败，13 个文件未格式化
  - 修复：格式化全部 `app/` 目录文件，修复 3 个未使用变量和 3 个导入排序问题
  - 影响文件：`apps/backend/app/` 下 15 个文件

### 测试 (Tests)

- L1 全量测试 99/101 通过（修复 AIASys-eval runner 的 workspace_dir 路径 bug 后）
- 前端首页多模态视觉检查通过
