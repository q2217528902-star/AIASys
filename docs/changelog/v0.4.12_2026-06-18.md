# Changelog v0.4.12 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **输入框发送后不清空**
  - 现象：用户发送消息后输入框保留原始文本，且后续可能无法继续发送
  - 根因：`handleSubmit` 中 `isSubmittingRef.current = false` 没有 try/finally 保护，`runAgentStream` 抛异常时 ref 永远保持 true，guard 拦截所有后续发送
  - 修复：用 try/finally 包裹函数体，确保 ref 始终被重置
  - 影响文件：`apps/web/src/pages/WorkspacePage/hooks/useCodeExecutor/useExecutionSubmit.ts`
