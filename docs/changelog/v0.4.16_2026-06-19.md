# Changelog v0.4.16 - 2026-06-19

## li-xiu-qi

### 新增功能 (Features)

- **StepFun Step Plan 流式 reasoning 兼容与模型级 reasoning_key**
  - 支持 StepFun Step Plan 模式下流式返回的 reasoning 内容解析
  - 在 `LLMProvider` 和 `LLMConfig` 中新增 `reasoning_key` 字段，允许按模型配置 reasoning 字段名
  - `OpenAIClient` 根据 provider 和模型自动选择正确的 reasoning 字段提取逻辑
  - `ProviderRouter` 在 Step Plan 和普通 API 之间正确路由并保留 reasoning 语义
  - 补充 `LLMConfigService` 对 reasoning_key 的序列化与读取
  - 影响文件：`apps/backend/app/models/llm_provider.py`、`apps/backend/app/services/agent/models/llm_config.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/backend.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/llm_clients/__init__.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/llm_clients/openai_client.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/provider_router.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/session_compaction.py`、`apps/backend/app/services/llm/llm_config_service.py`、`apps/backend/tests/test_aiasys_llm_clients.py`、`apps/backend/tests/test_llm_config_service.py`

- **开发环境端口占用自动探测与回退**
  - `dev.sh` 启动前后端服务时，若默认端口（13000/13001）被占用，自动探测并回退到可用端口
  - 后端 `app/main.py` 启动时读取 `AIASYS_BACKEND_PORT` 环境变量并打印实际监听端口
  - 新增 `app/core/port_utils.py`，提供 `find_available_port` 等跨平台端口探测工具
  - 影响文件：`scripts/dev/cli.sh`、`apps/backend/app/main.py`、`apps/backend/app/core/port_utils.py`

### 修复 (Bug Fixes)

- **修正 Windows 桌面版卸载时用户数据目录残留**
  - 问题：卸载向导选择「删除用户数据」后，`%APPDATA%\aiasys-desktop` 下的工作区文件、会话历史、本地数据库等依旧残留
  - 原因：用户数据目录名由 Electron app name（`aiasys-desktop`）决定，而卸载脚本错误地删除了带空格的 `AIASys Desktop`
  - 修改：`apps/desktop/build/installer.nsh` 中将删除目标修正为 `aiasys-desktop`，并新增 `SetShellVarContext current` 避免 per-machine 安装时 `$APPDATA` 指向 `ProgramData`
  - 同步修正 `apps/desktop/README.md` 中的 Windows 数据目录路径说明
  - 影响文件：`apps/desktop/build/installer.nsh`、`apps/desktop/README.md`

- **Shell 环境探测与 sqlite-vec 加载优化**
  - 修复 Shell 环境探测在某些场景下返回不稳定的问题
  - 优化 `sqlite_vec` 扩展加载逻辑，提升跨平台兼容性
  - 修复知识图谱前端相关组件的显示与交互问题
  - 同步更新若干 skill 脚本中的废弃引用与测试配置
  - 影响文件：`apps/backend/app/core/sqlite_vec.py`、`apps/backend/app/services/agent/config.py`、`apps/backend/app/services/agent/runtime_backends/acp_client/session.py`、`apps/backend/app/services/claw/adapters/status_shim.py`、`apps/backend/app/services/shell_environment.py`、`apps/backend/app/services/shell_executor.py`、`apps/backend/app/vendors/hermes_agent/hermes_constants.py`、`apps/backend/tests/conftest.py`、`apps/backend/tests/test_acp_client_session.py`、`apps/backend/tests/test_aiasys_llm_clients.py`、`apps/backend/tests/test_aiasys_runtime_backend.py`、`apps/backend/tests/test_llm_config_service.py` 等
