### 分离当前分支设置与工作区设置

#### 问题描述
- 现象：用户默认、当前分支覆盖和工作区级配置长期混在同一个设置面里，容易把作用域看混。
- 原因：分析页设置入口没有明确区分“当前分支”和“工作区”这两层语义。

#### 改动内容
- 新增分支级 Agent 配置覆盖与空分支 mode 更新逻辑。
- 把分析页设置明确拆成“当前分支设置”和“工作区设置”两条链路，相关对话框与 API 同步对齐。
