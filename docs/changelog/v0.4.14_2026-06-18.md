# Changelog v0.4.14 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **history.json 读写路径统一使用 session 目录**
  - 现象：会话历史文件 history.json 的读写路径与 session 实际目录不一致，可能导致历史恢复失败或数据丢失
  - 根因：历史文件路径解析逻辑未始终绑定到当前 session 目录
  - 修复：统一使用 session 目录作为 history.json 的读写根路径
  - 影响文件：`apps/backend/app/services/agent/mixins/session.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/session.py`、`apps/backend/app/services/agent/runtime_backends/base.py`

  - 影响文件：`apps/web/src/components/WorkspacePage/components/ResourceManagementDialog.tsx`
