# Changelog v0.4.15 - 2026-06-18

## li-xiu-qi

### 新增功能 (Features)

- **@ 文件引用功能**
  - 用户在输入框中可通过 `@/workspace/...` 或 `@/global/...` 引用工作区或全局文件
  - 消息气泡中美化显示为文件名标签，底层仍传递完整路径给后端
  - 输入框 textarea 上方显示已引用文件标签，点击可移除引用
  - 新增 `FileMentionPicker` 组件实现文件选择器
  - 影响文件：`apps/web/src/pages/WorkspacePage/components/FileMentionPicker.tsx`、`apps/web/src/pages/WorkspacePage/components/InputArea.tsx`、`apps/web/src/components/chat/ChatArea/UserMessageContent.tsx`、`apps/web/src/components/WorkspaceLayout/ConversationDock.tsx`、`apps/web/src/components/WorkspaceLayout/DockChatView.tsx`

- **全面强化 Windows 跨平台支持**
  - 新增 `AGENTS.md` 中 Windows 跨平台 Shell 策略、长路径处理、进程管理、编码处理等规范
  - 后端文件 IO 统一接入 `app/utils/path_utils.as_system_path()`，Windows 长路径自动加 `\\?\` 前缀
  - ACP Client 会话、`ShellExecutor`、`PTYManager`、`runtime_execution` 等关键路径全部接入长路径处理
  - Windows 子进程启动默认隐藏窗口，超时终止走 `taskkill /T` 两阶段策略
  - subprocess 输出统一走 `smart_decode()` 解码链，避免硬编码 `decode('utf-8')`
  - 新增设计文档：`design-draft/design/design-thinking/windows-cross-platform-shell-strategy.md`
  - 影响文件：`apps/backend/app/agents/tools/read_media_tool.py`、`apps/backend/app/agents/tools/shell_tool.py`、`apps/backend/app/api/routes/files_core.py`、`apps/backend/app/api/routes/files_utils.py`、`apps/backend/app/api/routes/terminal.py`、`apps/backend/app/services/agent/config.py`、`apps/backend/app/services/agent/runtime_backends/acp_client/session.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/backend.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/session.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/tools/monitor_tool.py`、`apps/backend/app/services/agent/subagent_lifecycle.py`、`apps/backend/app/services/agent/subagent_storage.py`、`apps/backend/app/services/container_resource.py`、`apps/backend/app/services/history/session_execution_journal.py`、`apps/backend/app/services/history/session_history_projection.py`、`apps/backend/app/services/node_runtime.py`、`apps/backend/app/services/runtime/runtime_execution.py`、`apps/backend/app/services/runtime_environment.py`、`apps/backend/app/services/shell_environment.py`、`apps/backend/app/services/shell_executor.py`、`apps/backend/app/services/terminal/pty_manager.py`、`apps/backend/app/storage/llm_provider_storage.py`、`apps/backend/tests/test_shell_executor.py`、`AGENTS.md`

- **桌面打包与 Windows Shell 修复**

  - 更新 `apps/desktop/BUILD_CHECKLIST.md`，补充桌面打包检查项

### 修复 (Bug Fixes)

- **彻底移除 cmd.exe 支持**
  - 现象：Windows 上 `interpreter=cmd` 或 `interpreter=auto` 时仍可能落到 cmd.exe，导致引号解析、WinError 267 等问题
  - 根因：`ShellExecutor` 中保留了对 cmd.exe 的识别与降级逻辑
  - 修复：完全删除 cmd.exe 相关分支，统一降级到 PowerShell；同步更新测试用例
  - 影响文件：`apps/backend/app/agents/tools/shell_tool.py`、`apps/backend/app/services/agent/runtime_backends/aiasys/session.py`、`apps/backend/app/services/shell_executor.py`、`apps/backend/tests/test_shell_executor.py`
