# Changelog v0.4.7 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **URL session_id 刷新后对话丢失**
  - 现象：用户通过 URL 访问工作区对话时，页面刷新后对话内容丢失，显示空状态
  - 根因 1：`navigateToAnalysisSession` 只写 `conversation_id` 不写 `session_id`，但 App.tsx 只读 `session_id`
  - 根因 2：`useWorkspaceRouteSync` 竞态条件——URL 中的 `session_id` 在 workspace conversation 列表加载完成前被替换
  - 修复 1：同时写入 `session_id` 和 `conversation_id` 到 URL
  - 修复 2：保留 URL 中的 `session_id` 不覆盖，让 bootstrap effect 尝试加载
  - 影响文件：
    - `apps/web/src/pages/WorkspacePage/hooks/useCodeExecutor/useSessionOrchestratorHelpers.ts`
    - `apps/web/src/pages/WorkspacePage/components/WorkspaceLayout/hooks/useWorkspaceRouteSync.ts`
