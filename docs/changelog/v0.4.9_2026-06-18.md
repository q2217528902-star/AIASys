# Changelog v0.4.9 - 2026-06-18

## li-xiu-qi

### 修复 (Bug Fixes)

- **NodeRuntimeEnv JSON 序列化错误**
  - 现象：RuntimeEnvironment 工具返回 Node 环境信息时抛出 `TypeError: Object of type NodeRuntimeEnv is not JSON serializable`
  - 根因：`_json_result` 用 `json.dumps` 序列化包含 Pydantic BaseModel 的 payload，但 BaseModel 不支持原生 JSON 序列化
  - 修复：添加 `default` 序列化器调用 `model_dump(mode='json')` 处理 Pydantic 对象
  - 影响文件：`app/agents/tools/runtime_environment_tool.py`
