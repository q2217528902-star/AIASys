# Changelog v0.2.11 - 2026-03-18

## li-xiu-qi

### 新增对话记录及工作区会话导出

#### 改动内容
- 后端新增会话导出接口 `/api/sessions/{user_id}/{session_id}/export`，支持 `conversation / workspace / bundle` 三种导出范围。
- 对话记录单独导出为 JSON，工作区单独导出为 ZIP，对话记录 + 工作区联合导出为单个审计 ZIP。
- 前端新增两个导出入口：当前会话顶部栏导出菜单、历史会话项更多菜单。
- 导出成功提示、文件名回退和按 scope 分流下载逻辑已接入前端主链路。

#### 安全边界
- 工作区导出默认排除 LLM provider 配置、API Key、`.env`、`config.local.json`、`settings.local.json`、`llm_config.json`、证书 / 私钥 / 密钥文件。
- 导出包不会包含 `.session/` 内部状态和运行时容器快照。
