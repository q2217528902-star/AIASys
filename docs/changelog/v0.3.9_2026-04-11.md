# Changelog v0.3.9 - 2026-04-11

> 说明：这是一次补记的主线汇总版本。`v0.3.3 ~ v0.3.8` 期间的内部迭代没有单独形成 release note，当前按用户可感知主线结果统一汇总到这里。

## li-xiu-qi

### 新增数据驱动 Scene Registry

#### 改动内容
- 后端引入 YAML 驱动的 Scene Registry，分析 / 研究等场景定义改成数据驱动发现，不再依赖硬编码路由分支。
- 新增 `/api/scenes` 与 `/api/scenes/{id}` 接口，为后续场景编排和入口收口提供统一事实源。

#### 文件变更
- `apps/backend/app/agents/scenes/_registry.yaml`
- `apps/backend/app/api/routes/scenes.py`
- `apps/backend/app/services/scene/registry.py`
- `apps/backend/app/services/agent/config.py`

### 收口为单一编排器 + research-enhancement 内置增强包

#### 问题描述
- 现象：分析 / 研究双模式长期把用户入口、内部能力组织和前端文案绑在一起，导致模式切换和工具增强语义混杂。
- 原因：研究能力仍被建模为独立 mode，而不是统一助手上的能力增强包。

#### 改动内容
- 后端把研究工具并入统一主配置，新增 `research-enhancement` 内置增强包。
- 前端移除 `ModeSelector`、研究模式选择卡和对应分叉文案，新建任务直接进入统一助手入口。
- 设计稿和主文档同步收口到“单一编排器 + 内置工具包”语义。

#### 文件变更
- `apps/backend/app/agents/local_sandbox_agent_config/data_analysis.yaml`
- `apps/backend/skills/builtin/research-enhancement/SKILL.md`
- `apps/backend/skills/builtin/research-enhancement/tools/paper_search.py`
- `apps/web/src/pages/DataAnalysisPage/components/ModeSelector.tsx`
- `apps/web/src/components/NewWorkspaceDialog/index.tsx`
- `apps/web/src/pages/DataAnalysisPage/components/TopBar.tsx`

### 移除 RESEARCH mode、Tavily MCP 和旧研究工具残留

#### 改动内容
- 后端删除 `RESEARCH` mode、研究路由和 Tavily MCP 集成，工作区 / Agent 配置字段统一回到单一 `analysis` 主线。
- 旧的论文下载、缓存与 OCR 工具拆成更清晰的 paper search / paper download / PaddleOCR 实现，不再维持旧研究模式残留。

#### 文件变更
- `apps/backend/app/api/routes/research.py`
- `apps/backend/app/models/workspace.py`
- `apps/backend/app/services/llm/mcp_config_service.py`
- `apps/backend/app/agents/tools/paper_search.py`
- `apps/backend/app/agents/tools/paper_download.py`
- `apps/backend/app/agents/tools/paddleocr_pdf_tool.py`

### 自动发现 builtin skill tools，并清理前端研究模式残留

#### 改动内容
- 后端新增 builtin skill tool 自动发现逻辑，运行时会自动扫描 `skills/builtin/*/tools/` 并注入到 Agent 配置。
- 前端同步把研究模式条件分支清到单一 `analysis` 类型，移除 AgentConfigPage 和分析页里遗留的研究模式 UI 条件。

#### 文件变更
- `apps/backend/app/services/agent/config.py`
- `apps/backend/app/agents/tools/paper_search.py`
- `apps/backend/app/agents/tools/paper_download.py`
- `apps/web/src/pages/AgentConfigPage/index.tsx`
- `apps/web/src/pages/DataAnalysisPage/hooks/useSessionManagement.ts`
